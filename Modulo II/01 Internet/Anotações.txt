Termos chaves:
cache:as informacoes que os sites deixam salvo no PC.

http: protocolo usado pela internet.
pixel: picture + element, os quadrados que compoem a imagem, cada um com a sua cor unidos geram a imagem que nós vemos.
URL: endereço que o link gerado carrega.

phishing: golpe que o golpista rouba dados enganando a outra pessoa, com emails, boletos, ligações, etc falsos.
cookies: histórico de pequenos registros após pesquisas salvos em nossa máquina enviado para outros sites com o intuito de ofertar produtos com base nas pesquisas anteriores.
firewall: parede que tem como função impedir acessos indevidos ao computador, mas contem muitas falhas, nem sempre sendo eficaz.

Stack Overflow: Forum para pesquisa e interação com outros desenvolvedores.

Aula 6

Stacks (conjunto, amontoado, pilhas(coletivo) em Inglês)
Tudo o que é necessário em nivel tecnológico para determinado objetivo (criar software, site, app, produzir uma TV, celular, etc)
As informações sobre a Stack vai definir a capacidade da equipe de desenvolvimento para concluir uma solução, estratégias de negócios, maturidade, contratações, mitigação de riscos, visão para aumento da capacidade, uso de dados.

Front-end: parte da frente, "cara/face" da solução. 
Aonde: no Site, Software, aplicativo, web service, interface do usuário, UI: User Interface, UX: User Experience.
Necessário conhecimento em: LP, HTML(LM), CSS (cores de estilo), jQuery/JS/AJAX (LP), PHP (LP), Boostrap/outros frameworks (biblioteca).

Back-end: parte de trás, bastidores da solução.
Aonde: servidores, banco de dados, faz o meio-de-campo entre interface e bancos de dados, regras de negócios, validações.
Necessário conhecimento em: MySQL(Banco de Dados), Oracle(Banco de Dados), protocolos de comunicação, PHP(LP), Java(LP), node.js(framework).

Full stack: profissional que atua como Front e Back end para desenvolvimento da Solução.

Quais são as tecnologias utilizadas na stack de grandes empresas (Google, Facebook, IBM, Apple, etc)?
